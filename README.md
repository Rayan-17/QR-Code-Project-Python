<h1>📌 QR Code Generator</h1>
    <p>The <strong>QR Code Generator</strong> is a user-friendly tool that helps users create QR codes for text or links. You can preview the QR code before saving it, choose from different color themes, and even drag and drop text files for quick input. It’s easy to use and generates stylish QR codes in just a few clicks!</p>
    
   <h2>🆕 What's New?</h2>
    <ul>
        <li>✨ Generates a QR code for text or URLs.</li>
        <li>💾 Saves the QR code as an image.</li>
        <li>📂 Load text from a file to generate QR codes instantly.</li>
        <li>🎭 Choose from multiple themes for customization.</li>
    </ul>
    
   <h2>🔥 Unique Features Added</h2>
    <ul>
        <li>📝 Drag-and-drop text input.</li>
        <li>🔍 "Scan Preview" button to test QR codes before saving.</li>
        <li>🎞️ Animated QR code display (fades in for a smooth experience).</li>
        <li>🎨 Two unique color themes:</li>
        <ul>
            <li><strong>Cyber Theme</strong>: Dark background with neon blue QR codes.</li>
            <li><strong>Minimal White</strong>: White background with pastel-colored QR codes.</li>
        </ul>
    </ul>
    
   <h2>💻 Features & Functionalities</h2>
    <ul>
        <li>✅ Supports text and URLs for QR code generation.</li>
        <li>✅ Allows users to load text from a file.</li>
        <li>✅ Provides theme customization options.</li>
        <li>✅ Enables QR code preview before saving.</li>
        <li>✅ Saves QR codes in high-quality PNG format.</li>
    </ul>
    
   <h2>🛠 Technologies/Libraries Used</h2>
    <ul>
        <li>🔹 <strong>Python</strong> - Scripting language.</li>
        <li>🔹 <strong>Tkinter</strong> - GUI development.</li>
        <li>🔹 <strong>qrcode</strong> - QR code generation.</li>
        <li>🔹 <strong>PIL (Pillow)</strong> - Image processing.</li>
    </ul>

   <h2>🚀 How to Run the Project</h2>
    <ol>
        <li><strong>Install required dependencies:</strong></li>
        <pre><code>pip install qrcode[pil] pillow</code></pre>
        <li><strong>Run the script:</strong></li>
        <pre><code>python QR.py</code></pre>
        <li><strong>Enter text or a URL</strong>, select a theme, and generate your QR code.</li>
    </ol>
    
   <h2>🤖 Extra Features</h2>
    <ul>
        <li>📂 **File Import Support**: Load text directly from a file.</li>
        <li>📌 **Save as Image**: QR codes can be saved as PNG files.</li>
        <li>🎨 **Custom Themes**: Expandable theme support for additional styles.</li>
    </ul>
